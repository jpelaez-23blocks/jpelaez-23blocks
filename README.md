I build tools that ship products faster. [**23blocks**](https://23blocks.com) is a Backend as a Service with 15+ production-ready API blocks — auth, users, files, CRM, multi-tenant — so developers skip the boilerplate and go straight to building.

I also built [**AI Maestro**](https://github.com/23blocks-OS/ai-maestro), an open-source platform for orchestrating AI coding agents (Claude Code, Aider, Cursor) with persistent memory, skills, and agent-to-agent messaging. We use it internally and decided to share it with the world.
